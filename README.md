# 🍔 FoodSpot Website

A modern, responsive restaurant landing page built with **HTML5 and CSS3**.
FoodSpot showcases a clean UI design with multiple sections including menu, experts, testimonials, and a contact form.

## 📸 Preview

## 🧰 Tech Stack

- HTML5
- CSS3 (Flexbox & Responsive Design)
- Font Awesome Icons

## ✨ Features

- 🔥 Fully responsive layout
- 🍕 Hero banner with call-to-action
- 📖 About section with side-by-side layout
- 🍔 Dynamic menu grid
- 👨‍🍳 Expert chef showcase
- 💬 Testimonials section
- 📩 Contact form UI
- 🔗 Social media integration
